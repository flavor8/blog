Permanent MIDI Names In ALSA

Every time you power cycle your PC or your MIDI devices, udev gives your MIDI devices new names. This can be annoying with some software. Here’s how to fix that.

Some software (e.g. Reaper under JACK) relies on the actual names given to the devices by udev/ALSA. Other software (e.g. Ardour, Bitwig, Reaper under ALSA) are more resilient, and instead rely on the actual USB device identifiers, which don’t change. If you’re using software of the former category, then you’ll need to edit a configuration file to force bind the devices to specific names.

Official documentation for doing this - long document, but not too bad to configure a small number of devices this way.
